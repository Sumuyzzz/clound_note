## 介绍

1. 创建账户
2. 登陆
3. 笔记本列表
4. 笔记本详情
5. 回收站

## 交互分析

1. 假设用户未登录时，访问网站，默认跳转到**登陆页面**
2. 用户此时可以登陆或者注册账号
3. 注册成功后切换登录页面
4. 登录成功后默认跳转到**笔记本列表**，用户可新建笔记本，删除笔记本，修改笔记本标题
5. 用户点击某条笔记本，会跳转到**笔记页面**，**笔记页面会展示**
6. 当用户点击回收站时，会跳转**回收站**页面，回收站里有临时删除笔记，用户可彻底删除笔记，也可撤掉操作。
7. 当用户点击注销，则回到登陆页面

## 准备工作
